Module: Enter the Cloud

Introduces how to seamlessly transfer your local machine setup to a cloud provider.

Learning Outcomes

Transfer to the Cloud

The student will get practical experience in how to set up a multi-machine environment that is able to seamlessly run both locally and on a cloud provider.

Experiential Learning

Create Cloud Account

Create an account on a cloud provider and set up Vagrant to use it

Multiple Machine Setup

Create a multi-machine setup for your cloud provider

Cloud Provisioning and Deployment

Provision your cloud multi-machine setup and deploy it on your cloud provider

Checkpoint

Experience Report

Summarise your workflow and your experiences

Checkpoint

Deploy Elsewhere

Deploy in the same way on Amazon Web Services

Optional

Assessments

Assessment for Entering the Cloud

Outcomes assessed: Transfer to the Cloud

You are done with this module when you have:

  • Created a multi-machine setup that runs on DigitalOcean.
  • Individually provisioned your machines using Puppet to run node.js and mysql, respectively.
  • Ensured that the machines are able to find and communicate with each other.

Assessment for Cloud Deployment Experience Report

Outcomes assessed: Transfer to the Cloud

You are done with this module when you have:

  • Summarised your experiences in an experience report.

Checkpoint for Cloud Deployment

Outcomes assessed: Transfer to the Cloud

Submit your experience report to a lab assistant and book a time. Be prepared to

  • discuss your experiences
  • show and answer questions about your provisioning
  • start up your boxes on DigitalOcean
