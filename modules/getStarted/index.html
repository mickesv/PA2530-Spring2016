Module: Getting Started With Virtualisation

Introduces a tool for virtualisation (VirtualBox), and a layer on top of this tool that enables automation (Vagrant).

Learning Outcomes

Install and Test VirtualBox

The student will be able to create a new virtual machine running Ubuntu.

Install and Test Vagrant

The student will be able to set up a basic ubuntu virtual machine with vagrant and ssh into it.

Assessments

Assessment for Getting Started with Virtualisation

Outcomes assessed: Install and Test VirtualBox, Install and Test Vagrant

You are done with this module when you have:

  • Installed VirtualBox
  • Created a virtual machine in VirtualBox
  • Installed an operating system on your virtual machine
  • Installed Vagrant
  • Created a Vagrant Box and logged in to it
  • Destroyed your started instance of a Vagrant Box
