Module: The Cloud Business Case

Different reasons for using the cloud. Select and describe a Project.

Learning Outcomes

Reasons for using Cloud Computing

The student will gain an understanding of different reasons for using cloud computing. The student will motivate the reasons for using a cloud solution for their specific project.

Select Project

The student will select a project to implementing a cloud solution for. The student will create an initial business case for this project.
