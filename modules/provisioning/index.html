Module: Provisioning

Introduces how to set up your virtual machine with the software you need in order to run your applications.

Learning Outcomes

Understand what provisioning is, and the need for it

The student will gain a basic understanding of what provisioning is, why it is needed, and some of the common tools available to set up your virtual machines.

Provisioning with Vagrant

The student will get practical experience in using the built-in capabilities of Vagrant to set up a virtual machine.
