Learning Outcomes

1. Install and Test VirtualBox

The student will be able to create a new virtual machine running Ubuntu.

Referencing modules: Getting Started With Virtualisation

Assessed by: Assessment for Getting Started with Virtualisation

2. Construct Foos

The student will be able to quickly and correctly construct instances of Foos.

Referencing modules: Learn to Foo

Assessed by: Assessment of Foo construction skill

3. Understand what provisioning is, and the need for it

The student will gain a basic understanding of what provisioning is, why it is needed, and some of the common tools available to set up your virtual machines.

Referencing modules: Provisioning

Assessed by: Assessment for Provisioning

4. Provisioning with Vagrant

The student will get practical experience in using the built-in capabilities of Vagrant to set up a virtual machine.

Referencing modules: Provisioning

Assessed by: Assessment for Provisioning

5. Install and Test Vagrant

The student will be able to set up a basic ubuntu virtual machine with vagrant and ssh into it.

Referencing modules: Getting Started With Virtualisation

Assessed by: Assessment for Getting Started with Virtualisation
