Learning Outcomes

1. Install and Test VirtualBox

The student will be able to create a new virtual machine running Ubuntu.

Referencing modules: Getting Started With Virtualisation

Assessed by: Assessment for Getting Started with Virtualisation

3. Understand what provisioning is, and the need for it

The student will gain a basic understanding of what provisioning is, why it is needed, and some of the common tools available to set up your virtual machines.

Referencing modules: Provisioning

Assessed by: Assessment for Provisioning

4. Transfer to the Cloud

The student will get practical experience in how to set up a multi-machine environment that is able to seamlessly run both locally and on a cloud provider.

Referencing modules: Enter the Cloud

Assessed by: Assessment for Entering the Cloud

5. Setting up a Multi-Machine environment with Vagrant

The student will get practical experience in using Vagrant to set up an environment consisting of several heterogeneous machines.

Referencing modules: Multiple Machines

Assessed by: Assessment for Multiple Machines

6. Lookup Service

The student will get practical experience in how to set up a multi-machine environment that is aware of how to reach the other machines in the configuration.

Referencing modules: Multiple Machines

Assessed by: Assessment for Multiple Machines

7. Provisioning with Vagrant

The student will get practical experience in using the built-in capabilities of Vagrant to set up a virtual machine.

Referencing modules: Provisioning

Assessed by: Assessment for Provisioning

8. Install and Test Vagrant

The student will be able to set up a basic ubuntu virtual machine with vagrant and ssh into it.

Referencing modules: Getting Started With Virtualisation

Assessed by: Assessment for Getting Started with Virtualisation
